Tipos de selectores que entran en el exámen

Clase (class="")

Selecciona todos los elementos que tengan la misma clase. Este es un ejemplo de un párrafo de clase "ejemplo1" al que se le ha designado el color magenta.

Identificador (id="")

Selecciona el elemento con una id específica. Este párrafo tiene la id "ejemplo2", por lo que es el único párrafo con estas cualidades.

Todos los elementos (*)

Selecciona todos los elementos, es decir, le aplica a todos los elementos el mismo estilo. En esta web, se ha aplicado el mismo color de fondo a todos los elementos.

Varios elementos (ul, li, h1)

Selecciona todos los elementos dentro del paréntesis. En este ejemplo se ha seleccionado (h1,ul y li), de manera que todos estos elementos tengan el mismo estilo.

Elementos contenidos en otros elementos (ol li)

Selecciona todos los elementos "li" dentro de la lista ordenada "ol".

Sobrevuelo (p:hover)

Este selector te permite cambiar el estilo cuando pones el cursor encima de un elemento, todos los párrafos de esta web, menos el 2º, cambian al color amarillo cuando pasas el cursor por encima.
